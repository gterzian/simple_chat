##### How to run:

1. `curl -sSf https://static.rust-lang.org/rustup.sh | sh`
2. `git clone git@github.com:gterzian/simple_chat.git`
3. `cd simple_chat`
4. `cargo test`

Alternatively, just look at it running on [travis](https://travis-ci.org/gterzian/simple_chat)...
